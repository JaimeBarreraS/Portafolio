Adivina La Carta
El juego selecciona aleatoriamente una carta de referencia y el usuario debe predecir si la próxima carta será mayor, menor o igual. Se utiliza la API de Deck of Cards para obtener cartas aleatorias en cada ronda.

Tecnologias

Tecnologías Utilizadas

Características

Características Principales
Interactivo: Permite adivinar si la siguiente
carta es mayor, menor o igual.

Interfaz: Diseño limpio y atractivo con
estilos CSS personalizados.

Uso de API: Se conecta con la
API Deck of Cards para obtener cartas
aleatorias en cada partida.

Respuestas dinámicas: En tiempo real sobre
si la respuesta del jugador es correcta
o incorrecta.

Botón de reinicio: Permite empezar una nueva
partida fácilmente.
